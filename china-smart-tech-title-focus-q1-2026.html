Filter rule: The China report no longer keys off the single word `智能`. It now uses two layers: `App / Wi-Fi / Bluetooth / voice / ecosystem / remote connectivity / tracking / monitoring` are hard cues and count directly; `robot` only counts in native cleaning, monitoring, or tracking contexts; `smart / AI / smart control` are soft cues and count only when paired with a hard cue. High-noise words such as `touch / automatic / timer / constant temperature / display / visual`, plus leaf accessory/consumable buckets, are tracked only as excluded observations.
