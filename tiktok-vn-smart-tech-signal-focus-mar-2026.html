🧠 智能词逻辑🧠 Smart Cue Logic

筛选原则：本报告只把“硬信号”直接纳入，如 `app / wifi / bluetooth / giọng nói`；`smart / thông minh`、`cảm biến`、`điều khiển từ xa` 这些“软信号”只有在同时伴随硬信号时才算智能，否则会被打到观察池。这样可以明显减少 `Mini Smart`、`thông minh` 这类营销型命名误判，尤其是 `Electric Kettles`。
Filter rule: Only `hard cues` are accepted directly, such as `app / wifi / bluetooth / voice`. `Soft cues` like `smart / thong minh`, `sensor`, and `remote control` count only when they appear together with a hard cue; otherwise they are pushed into the observed-but-excluded pool. This sharply reduces false positives from marketing-style naming, especially in `Electric Kettles`.
